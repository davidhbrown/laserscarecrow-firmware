Be certain to select the correct model firmware! If you try to use the wrong firmware, it won't work (and you might damage something).

2022 Model - circular green main board

This is an updated firmware (1.3.0) that has an improved algorithm for random motion when the active area is small. Useful if, for example, you position the scarecrow at the corner of a field.

2023 Model - circular blue main board

This the originally released firmware (2.0.0).

Replacing the ESP32-DevKitC microcontroller module

The microcontroller in these modules is the Espressif ESP32-DEVKITC-VIE. We have purchased them from...

  • Digikey
  • Mouser

You should be able to substitute the ESP32-DevKitC-VE. (The only difference is in the antenna which we don't use.) While these are still (Aug 2023) current products, be careful if you search for a replacement online (e.g., Amazon, eBay) as you're may find more recent variations such as an ESP32-S2/S3/C3 that aren't quite the same.
